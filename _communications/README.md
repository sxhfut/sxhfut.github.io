# MAC-Lab Communication Kit

This folder contains draft communication materials for MAC-Lab. It is a Jekyll underscore folder, so it is not published as a normal website page by default.

## Suggested Use

- Use `01-feature-article-cn.md` for Chinese news sites, WeChat public account, Zhihu column, Baijiahao, Sohu, NetEase, Toutiao, and university or partner news channels.
- Use `02-feature-article-en.md` for LinkedIn, Medium, ResearchGate, international collaborator updates, and English website reposting.
- Use `03-platform-adaptations.md` for Xiaohongshu, WeChat, Zhihu, LinkedIn, and short social posts.
- Use `04-submission-email.md` when contacting media, institutional channels, or partner communication teams.

## Transparency Note

Use transparent sourcing such as "MAC-Lab", "Professor Xiao Sun's team", "Hefei University of Technology MAC-Lab", "contributed by the lab", or a real partner channel. Do not present lab-written content as independent third-party reporting unless a real editor or media organization has independently reviewed and published it.
